# UI для имитационной модели планирования ресурсов мобильной сети

## Описание проекта
Проект посвящён разработке **пользовательского интерфейса** для управления и настройки имитационной модели планирования радиоресурсов мобильной сети (5G/LTE).

Основная цель — предоставить удобный способ конфигурировать параметры симуляции:
- Поведение пользовательских устройств (UE)
- Параметры базовой станции (BS)
- Модели радиоканала
- Источники трафика
- Модели передвижения
- Алгоритм планировщика (Scheduler)

Пользовательский интерфейс служит **надстройкой** над будущей симуляционной системой, позволяя формировать входные данные для моделирования без необходимости менять код вручную.

## Цель проекта
Создать человеко-ориентированный интерфейс, который позволит:

- Формировать структуру симуляции  
- Гибко настраивать параметры моделей  
- Управлять сущностями (UE, BS)  
- Экспортировать параметры в пригодный формат
- Тестировать, визуализировать и сравнивать результаты работы алгоритмов планирования ресурсов  

## Основной функционал
- Создание и настройка User Equipment (UE)
- Настройка параметров Base Station (BS)
- Выбор и конфигурация mobility-моделей
- Настройка моделей канала (RMa, UMa, UMi)
- Настройка трафиковых моделей
- Выбор планировщика ресурсов
- Навигация через боковую панель с иконками
- Подготовка данных для дальнейшего моделирования
